Siamo quasi alla fine del corso CSS, e in questo video l’esempio che uso è Twitter. Perché proprio Twitter?  
Perché è un caso perfetto per mostrare come JavaScript si inserisce nel lavoro fatto finora con HTML e CSS.

HTML è il contenuto: i bottoni, gli input, i titoli come `h1`, i paragrafi, gli attributi di accessibilità come `aria-label`, i form. Tutto ciò che abbiamo già studiato nelle lezioni precedenti.

CSS si occupa invece dell’aspetto grafico: lo sfondo nero, il colore del testo, i font, i gradienti, i bordi arrotondati, le animazioni, i link che cambiano colore al passaggio del mouse, le media query che adattano la visualizzazione su mobile. Insomma, la parte estetica.

E allora cos’è JavaScript?  
JavaScript è ciò che dà vita e azione al sito.

Un esempio evidente è lo spinner iniziale di Twitter: appena aggiorni la pagina, la struttura si carica subito, ma i dati arrivano dopo. Quei dati vengono richiesti tramite JavaScript, e sempre tramite JavaScript vengono inseriti dinamicamente nella pagina. Lo spinner non è un elemento nascosto nell’HTML: viene proprio creato e mostrato da JavaScript.

Anche quando apri un pannello, come il menu impostazioni o un dialog che si apre sopra la pagina, spesso dietro c’è JavaScript. È vero che con le nuove specifiche HTML e CSS possiamo creare alcuni di questi comportamenti anche senza script, ma per ottenere interazioni più avanzate serve comunque JavaScript.

Un altro esempio: quando nelle impostazioni premi “indietro”. Non c’è un vero link che porta a un’altra pagina, ma un’azione JavaScript che aggiorna l’interfaccia.  
Oppure il carosello di immagini che abbiamo visto nel sito WordPress che avevamo copiato. Quell’effetto è tipico JavaScript.

Un generatore di password? Anche quello funziona grazie a JavaScript: CSS non può generare nulla, HTML può solo mostrare dati già esistenti, ma il calcolo lo fa JavaScript. Tutto ciò che viene creato, modificato, aggiunto o rimosso dinamicamente da una pagina è realizzato con JavaScript.

In sintesi:

- **HTML** è il contenuto.
    
- **CSS** è l’aspetto visivo.
    
- **JavaScript** è il movimento, l’azione, l’interattività.
    

È il terzo pilastro di un sito moderno. Finora hai costruito la struttura (HTML) e l’estetica (CSS). Ora serve JavaScript per dare vita e dinamicità al tutto.
