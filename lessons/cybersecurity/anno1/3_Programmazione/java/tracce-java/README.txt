
BENVENUTO! Se stai leggendo questo README, vuol dire che probabilmente
hai appena finito di studiare la sintassi di base di Java e vuoi
mettere in pratica ciò che hai imparato con gli esercizi qui presenti.

Ho sviluppato tre categorie di esercizi, in ordine di difficoltà crescente,
che sono raggruppati in tre cartelle distinte:

E1_esercizi_introduttivi  --> esercizi di base per prendere confidenza con la sintassi di Java
E2_gestionali             --> esercizi di media difficoltà che iniziano a far spremere le meningi
E3_simulazione_giochi     --> esercizi più complessi che simulano piccoli giochi o applicazioni da terminale

Ogni cartella contiene a sua volta delle sottocartelle, ognuna delle quali
contiene un singolo esercizio e un relativo file README.txt con le note più rilevanti che
non volevo mettere inline nel codice sorgente per mantenere un'alta leggibilità.
